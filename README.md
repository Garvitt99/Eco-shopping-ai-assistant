# Eco shopping ai assistant – Responsible Consumption & Production

## Overview
This project is an AI-powered assistant designed to support responsible consumption by helping users make sustainable purchasing decisions.

The bot provides:
- Eco-friendly product awareness
- Waste reduction guidance
- Recycling and repurposing suggestions
- Automated sustainability query handling through email interaction

## Features
- AI-based sustainable product recommendation logic
- Automated first-response email system
- Guidance for reducing waste and promoting responsible consumption
- User query handling through workflow automation

## Problem Statement
The goal of this project is to promote sustainable consumption by helping users choose environmentally responsible products and adopt better waste management habits.

## Technology Used
- Relay.app workflow automation
- AI prompt logic
- Email automation
- Decision-based response handling
